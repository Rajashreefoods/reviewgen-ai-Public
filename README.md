# Rajashree Foods ReviewGen AI V1.2

Static GitHub Pages version for Rajashree Foods.

## Correct GitHub Pages URL

Because the repository shown in GitHub Pages settings is named `reviewgen-ai-Public`, the project site URL should be:

https://rajashreefoods.github.io/reviewgen-ai-Public/

If you want the URL to be:

https://rajashreefoods.github.io/reviewgen-ai/

rename the GitHub repository to `reviewgen-ai`, or create/upload these files into a repository with that exact name.

## Files to upload to repository root

Upload these files directly into the root of the `main` branch:

- `index.html`
- `style.css`
- `app.js`
- `logo.png`
- `.nojekyll`
- `README.md`

## GitHub Pages settings

Use:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/ (root)`
